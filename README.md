# bltui 
 
---

## Installation

### Cargo

`cargo install bltui`

### AUR

`pacman -S bltui`

## Usage

 - `-d/-dd`: enable debug/trace log level. Recommended to use file logging with it since logger output is small.
 - `-l`: log to file (`$HOME/.bltui/logs`)
